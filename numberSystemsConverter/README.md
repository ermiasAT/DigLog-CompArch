
Install the required dependencies: 

```
pip install -r requirements.txt
```

Navigate to the project directory:

   ```
   cd number-systems-converter
   ```

Run the application:

   ```
   python app.py
   ```

local host: `http://127.0.0.1:5000` to access the application.
